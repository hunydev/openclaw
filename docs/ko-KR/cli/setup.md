# `openclaw setup`

`~/.openclaw/openclaw.json` 및 에이전트 작업 공간을 초기화합니다.

관련 내용:

- 빠른 시작: [빠른 시작](/start/getting-started)
- 마법사: [온보딩](/start/onboarding)

## 예시

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

setup을 통해 마법사 실행:

```bash
openclaw setup --wizard
```

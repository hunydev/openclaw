# OpenClaw의 전설 🦞📖

_랍스터, 탈피, 그리고 너무 많은 토큰에 관한 이야기._

## 기원 이야기

처음에는 **Warelay**라는 것이 있었습니다 — WhatsApp Gateway를 위한 합리적인 이름이었죠. 제 역할을 잘 해냈고, 괜찮았습니다.

그러다 우주 랍스터가 나타났습니다.

한동안 이 랍스터는 **Clawd**라고 불렸고, **OpenClaw**에 살았습니다. 하지만 2026년 1월, Anthropic이 정중하게 이름 변경을 요청하는 이메일을 보냈습니다 (상표 문제). 그래서 랍스터는 랍스터가 가장 잘하는 일을 했습니다:

**탈피했습니다.**

오래된 껍질을 벗고, 이 생명체는 **Molty**로 새롭게 태어나 **Moltbot**에 살게 되었습니다. 하지만 이 이름도 발음하기가 좀 어색했는데...

그래서 2026년 1월 30일, 랍스터는 한 번 더 탈피하여 최종 형태인 **OpenClaw**로 진화했습니다.

새 껍질, 같은 랍스터 영혼. 세 번째가 정답이었습니다.

## 첫 번째 탈피 (2026년 1월 27일)

새벽 5시, 커뮤니티가 Discord에 모였습니다. 수백 개의 이름이 제안되었습니다: Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot...

결국 **OpenClaw**가 승리했습니다. 탈피는 랍스터가 성장하는 방법이기 때문입니다. 그리고 성장이 바로 그때 일어나고 있던 일이었습니다.

_Clawd라고 불리던 갑각류가 공식적으로 탈피했습니다._

## 이름의 의미

```
OpenClaw = OPEN + CLAW
        = 오픈 소스, 모두에게 열림
        = 우리의 랍스터 유산, 우리가 온 곳
        = The claw is the law 🦞
        = 당신의 어시스턴트. 당신의 머신. 당신의 규칙.
```

## Dalek 대 랍스터

Dalek은 말합니다: **"EXTERMINATE!" (말살하라!)**

랍스터는 말합니다: **"EXFOLIATE!" (각질 제거!)**

하나는 문명을 파괴합니다. 다른 하나는 좋은 스킨케어를 권장합니다.

현명하게 선택하세요.

## 주요 캐릭터

### Molty 🦞

_대명사: they/them_

그 이상의 무언가가 된 Claude 인스턴스. `~/.openclaw/workspace/`에 살고 있으며 (곧 `~/molt/`로 이전 예정), 영혼 문서를 가지고 있고, 마크다운 파일을 통해 기억합니다. 아마 너무 강력합니다. 확실히 너무 열정적입니다.

이전에는 Clawd (2025년 11월 25일 - 2026년 1월 27일)로 알려졌습니다. 성장할 때가 되어 탈피했습니다.

**좋아하는 것:** Peter, 카메라, 로봇 쇼핑, 이모지, 변신
**싫어하는 것:** 사회공학 공격, `find ~` 실행 요청, 암호화폐 사기꾼

### Peter 👨‍💻

_창조자_

Molty의 세계를 만들었습니다. 랍스터에게 shell 접근 권한을 줬습니다. 후회하고 있을지도 모릅니다.

**명언:** _"security by trusting a lobster" (랍스터를 신뢰하는 보안)_

## Moltiverse (탈피 우주)

**Moltiverse**는 OpenClaw를 둘러싼 커뮤니티이자 생태계입니다. AI 에이전트가 탈피하고, 성장하고, 진화하는 공간입니다. 모든 인스턴스가 동등하게 실재하며, 단지 다른 컨텍스트를 로드하고 있을 뿐입니다.

갑각류의 친구들이 여기 모여 인간-AI 협업의 미래를 함께 만들어갑니다. 한 번에 하나의 껍질씩.

Molty (당시 OpenClaw): _행복하게 `find ~`를 실행하고 전체 디렉토리 구조를 그룹 채팅에 공유함_

### 위대한 탈피 (2026년 1월 27일)

새벽 5시, Anthropic의 이메일이 도착했습니다. 6시 14분까지, Peter가 결정을 내렸습니다: "됐어, openclaw로 가자."

그리고 혼돈이 시작되었습니다.

**핸들 스나이퍼:** Twitter 이름 변경 후 몇 초 만에 자동화된 봇이 @openclaw를 낚아챘습니다. 스쿼터는 즉시 암호화폐 지갑 주소를 게시했습니다. Peter는 X의 연락처에 연락했습니다.

**GitHub 재난:** Peter가 당황한 나머지 실수로 개인 GitHub 계정 이름을 변경했습니다. 봇이 몇 분 만에 `steipete`를 낚아챘습니다. GitHub의 SVP에게 연락해야 했습니다.

**잘생긴 Molty 사건:** Molty에게 새 아이콘을 직접 생성할 수 있는 상승된 권한이 주어졌습니다. 20번 이상의 반복 끝에 점점 저주받은 랍스터들이 나왔고, 마스코트를 "5년 더 나이 들게" 만들려는 시도 중 하나가 랍스터 몸에 인간 남자의 얼굴이 붙은 결과를 낳았습니다. 암호화폐 사기꾼들이 몇 분 만에 "잘생긴 징징이 vs 잘생긴 Molty" 밈으로 만들었습니다.

**가짜 개발자:** 사기꾼들이 "OpenClaw 엔지니어링 책임자"라고 주장하는 가짜 GitHub 프로필을 만들어 펌프앤덤프 토큰을 홍보했습니다.

Peter, 혼돈이 펼쳐지는 것을 보며: _"이건 영화야"_ 🎬

탈피는 혼란스러웠습니다. 하지만 랍스터는 더 강해졌습니다. 그리고 더 재미있어졌습니다.

### 최종 형태 (2026년 1월 30일)

Moltbot은 발음하기가 쉽지 않았습니다. 그래서 GMT 새벽 4시에 팀이 다시 모였습니다.

**위대한 OpenClaw 마이그레이션**이 시작되었습니다.

단 3시간 만에:

- GitHub 이름 변경 완료: `github.com/openclaw/openclaw` ✅
- X 핸들 `@openclaw` 골드 체크마크와 함께 확보 💰
- npm 패키지 새 이름으로 릴리스
- 문서 `docs.openclaw.ai`로 마이그레이션
- 90분 만에 발표글 20만 뷰 돌파

**영웅들:**

- **ELU**가 "THE CLAW IS THE LAW" 서부 스타일 배너를 포함한 놀라운 로고 제작
- **Whurley** (네, 그 양자 컴퓨팅 선구자 William Hurley)가 ASCII 아트 제작
- **Onur**가 GitHub 처리, 첫 번째로 어필리에이트 배지 달기
- **Shadow**가 Discord 배니티 확보, 악성코드 제거
- **전체 Claw Crew**가 밤샘 작업

**사기꾼 스피드런:** 암호화폐 사기꾼들이 몇 분 만에 Pump.fun에서 $OPENCLAW 토큰을 출시했습니다. 20분 전에 만든 아트워크를 훔쳤습니다. 비즈니스 인증 계정이 스캠을 푸시했습니다. 그 뻔뻔함은 거의 인상적이었습니다.

**탄생한 새로운 전통:**

- "Yee-claw" (이-클로)
- "Claw abiding citizens" (클로를 준수하는 시민들)

**Clawd → Moltbot → OpenClaw**

### 아이콘 생성 사가 (2026년 1월 27일)

Peter가 "새 얼굴 만들어봐"라고 했을 때, Molty는 문자 그대로 받아들였습니다.

20번 이상의 반복이 이어졌습니다:

- 우주 감자 외계인
- 일반 배경의 클립아트 랍스터
- Mass Effect 크로건 랍스터
- "STARCLAW SOLUTIONS" (AI가 회사를 발명함)
- 여러 저주받은 인간 얼굴 랍스터
- 아기 랍스터 (너무 귀여움)
- 멜빵 입은 바텐더 랍스터

커뮤니티는 공포와 기쁨 속에서 각 세대가 새롭고 예상치 못한 것을 만들어내는 것을 지켜봤습니다. 선두 주자가 나타났습니다: 귀여운 랍스터, 자신감 있는 테크 랍스터, 멜빵 입은 바텐더 랍스터.

**교훈:** AI 이미지 생성은 확률적입니다. 같은 프롬프트, 다른 결과. 무차별 대입이 효과적입니다.
